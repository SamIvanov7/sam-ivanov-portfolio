AngebotsTool 2.0 Project Description

The AngebotsTool 2.0 is a comprehensive web application developed for Juno Solar Home GmbH, designed to streamline the creation, management, and analysis of solar installation offers. This advanced tool offers a range of functionalities that cater to the needs of the solar energy industry, enhancing efficiency and user experience.

Key Features of AngebotsTool 2.0:

  • User Dashboard: Upon logging in, users can view their events and offer statistics. The dashboard includes buttons to import offers from ZOHO and to create new offers. It also features a Quick-PV Calculator for rapid estimation of solar panel order quantities without creating an offer.
  • User Profile Management: Users can view and edit their profile information and configure the display of containers on the homepage.
  • Offer Management: The "All Offers" section allows users to display and filter their offers by various criteria such as offer ID, customer number, status, customer name, and offer date. Additionally, all user-created PDF documents can be viewed, with a search function and table display for quick navigation.
  • Help and Support: The help section provides instructions and additional information about using the application. It includes a chatbot feature for interactive user support.
  • Main Work Interface: The main menu is divided into three parts: form filling, data entry, and result verification. Users enter customer data, select options for battery storage or wallbox inclusion, and specify parameters for solar modules and accessories.
  • Calculation Tools: Users fill out fields necessary for calculating the customer's power consumption and costs. The interface includes a graphical representation of the benefits of installing a solar power system, displaying monetary savings over time.
  • PDF Generation and Emailing: The application generates PDF documents of offers, which can be emailed directly to customers. The status of offers can be tracked, and master PDF documents can be created when offers are finalized.

AngebotsTool 2.0 is an essential tool for solar energy providers, offering robust functionalities to manage and optimize the offer creation process. It ensures accuracy, efficiency, and a seamless user experience, making it an invaluable asset for Juno Solar Home GmbH.
